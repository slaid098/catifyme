# CatifyMe

> Upload a selfie — get a cartoon cat that matches your vibe. 🐱

A mobile-first web app that turns your selfie into a cartoon cat character
with a name and personality. Powered by Puter.js (user-pays model — free for
the developer, users cover their own AI usage via a free Puter tier).

## Status

🚧 In development — see [project board](../../projects) and commit history.

## Stack

- Vanilla HTML / CSS / JS (no build step, no framework)
- [Puter.js](https://developer.puter.com/) for AI vision + image generation
- Mobile-first, dark theme, RU/EN i18n with auto-detect
- Hosted on Vercel — `catifyme.vercel.app`
